# Homebrew Tap for Pesto Clipboard

## Installation

```bash
brew install matthewpick/pesto-clipboard/pesto-clipboard
```

Or:

```bash
brew tap matthewpick/pesto-clipboard
brew install pesto-clipboard
```

## About

[Pesto Clipboard](https://github.com/matthewpick/pesto-clipboard) is a clipboard manager for macOS.
